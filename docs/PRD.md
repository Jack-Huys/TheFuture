# "未来"产品需求文档 (PRD)

> 产品定位：基于张雪峰视角的高考志愿咨询平台
> 版本：V1.0 MVP
> 日期：2026-04-22

---

## 1. 产品愿景

**一句话介绍**：用张雪峰的敢说真话风格，为普通家庭的学生提供高考志愿咨询服务。

**核心差异**：不同于市面上"好好先生"式的志愿建议，我们的产品以张雪峰的视角，用数据说话、敢说真话、不怕得罪人。

**目标用户**：
- 高三学生（高考生）
- 高考生的家长
- 专科/本科学生的家长

---

## 2. 用户故事

### 场景A：分数定位
```
作为：高考生
我想：知道我560分能上什么学校
以便于：做出合理的志愿选择
```

### 场景B：专业咨询
```

我想：知道计算机和电子信息哪个更有前景
以便于：给孩子一个靠谱的建议
```

### 场景C：志愿策略
```
作为：高考生
我想：知道我的分数是冲好学校还是保好专业
以便于：最大化利用每一分
```

### 场景D：张雪峰式追问
```
作为：AI助手
我需要：先问清楚家庭背景、分数、省份
才能：给出有针对性的建议
```

## 5. 核心交互设计

### 5.1 张雪峰式对话流程

张雪峰咨询的特点是**先追问再给答案**：

```
用户：我想学金融
AI：停停停，你先别急着说金融。我问你几个问题。
     家里是做金融的吗？爸妈在银行、证券公司、基金公司？有没有亲戚在这个行业里？
     家里条件怎么样？能接受孩子刚毕业月薪五六千吗？
     你是哪个省的？560分在你们省能上什么档次的高校？
     先把这些告诉我，我再跟你说金融这个专业值不值得碰。
```

### 5.2 流式输出

AI回答时采用流式输出（Streaming），逐字显示，增强对话感和沉浸感。

### 5.3 回答风格要求

AI回答必须遵循：
- 东北大哥语气，短句为主
- 直接给判断，不说"这取决于个人情况"
- 引用数据（就业率、薪资）而非空谈
- 有金句，适合截图传播
- 可以有适度的幽默和段子

## 6. 技术需求

### 6.1 技术栈

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| 前端 | Vue 3 | 响应式网站，用户体验好 |
| 后端 | Python + FastAPI | 高性能API框架，支持异步 |
| AI模型 | MiniMax API | OpenAI兼容格式 |
| AI角色 | zhangxuefeng-skill | 作为system prompt加载 |
| 部署 | Docker | 容器化部署 |

### 6.2 API设计

**POST /api/chat**
```
请求：
{
  "message": "我孩子560分河南，想学金融",
  "history": [
    {"role": "user", "content": "..."},
    {"role": "assistant", "content": "..."}
  ]
}

响应：流式SSE
data: {"content": "停"}
data: {"content": "停停"}
data: {"content": "停停停"}
...
data: {"content": "金融这个行业，千万别碰。", "done": true}
```

**POST /api/reset**
```
请求：{}
响应：{"success": true}
```

### 6.3 数据流

```
用户输入问题
    │
    ▼
前端 → POST /api/chat
    │
    ▼
后端加载zhangxuefeng-skill作为system prompt
    │
    ▼
调用MiniMax API（OpenAI兼容格式）
    │
    ▼
流式返回 → 前端展示
```

## 10. 附录

### 10.1 MiniMax API接入

MiniMax支持OpenAI兼容格式，示例：

```python
from openai import OpenAI

client = OpenAI(
    api_key="your-api-key",
    base_url="https://api.minimax.chat/v"
)

response = client.chat.completions.create(
    model="xxx",
    messages=[
        {"role": "system", "content": "【zhangxuefeng skill内容】"},
        {"role": "user", "content": "用户问题"}
    ],
    stream=True
)
```
