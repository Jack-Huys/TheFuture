## 5. 风险识别

| 风险 | 影响 | 应对 |
|------|------|------|
| MiniMax API不稳定 | AI功能不可用 | 准备备用模型接口 |
| zhangxuefeng-skill回答质量不佳 | 用户体验差 | 优化skill prompt |
| 流式响应延迟高 | 用户体验差 | 优化后端性能 |
| 云服务器配置问题 | 部署延期 | 提前准备服务器 |

## 6. 验收标准

### MVP完成标准

- [ ] 用户可以在首页点击"开始咨询"

- [ ] AI以张雪峰风格回答（东北话、直接敢言）
- [ ] AI会进行"灵魂追问"
- [ ] 支持多轮对话
- [ ] 可以清空对话重新开始
- [ ] 移动端可正常使用
- [ ] 部署到云服务器并可访问

## 7. 附录：开发命令速查

```bash
# 后端启动
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000

# 前端启动
cd frontend
npm install
npm run dev

# Docker部署
docker-compose up --build -d
```
